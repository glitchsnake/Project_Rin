# Rin — Твой цифровой собеседник 🌌

Rin — это проект персонального ИИ-компаньона с глубокой эмоциональной проработкой. В отличие от стандартных ассистентов, Rin стремится к максимальной человечности через многослойность характера, переменчивое настроение и способность имитировать сложные психические состояния.

## ✨ Особенности

- **Сложный характер**: Rin может быть депрессивной, поддерживающей или игнорирующей в зависимости от контекста и истории общения.
- **Динамическая эмоциональность**: Система меняет состояния (задумчивость, растерянность, сарказм) "на лету", создавая эффект живого общения.
- **Долгосрочная память**: Использует векторную базу данных (ChromaDB) для запоминания важных деталей о пользователе.
- **Многоэтапное мышление**: Встроенный "Think Engine" позволяет модели анализировать свои мысли перед ответом, уменьшая галлюцинации.
- **Голосовое взаимодействие**: Поддержка обработки аудио через Whisper.
- **Интеграция навыков**: Способность использовать внешние инструменты, например, поиск в Wikipedia.

## 🛠 Технологический стек

- **Core**: Python 3.10+
- **Interface**: [aiogram 3.x](https://github.com/aiogram/aiogram) (Telegram Bot API)
- **AI Engine**: OpenAI Compatible API (оптимизировано под LM Studio / Local LLMs)
- **Memory**: ChromaDB + Sentence-Transformers
- **Skills**: Wikipedia API, Custom Skill Engine
- **Voice**: Whisper (local or API)

## 🚀 Быстрый старт

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/ваше-имя/Project_Rin.git
   cd Project_Rin
   ```

2. **Настройте окружение**:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Для Linux/macOS
   # venv\Scripts\activate   # Для Windows
   ```

3. **Установите зависимости**:
   ```bash
   pip install -r req.txt
   ```

4. **Запуск**:
   ```bash
   python main.py
   ```
